Github Pagesで自分のWebサイトを公開することができます！

やったこと
- Cloudflareで独自ドメインを取得(やらなくても良いしやっても良い)
    - CloudflareでDNS管理も行う
- Github Pages
    - sfujibijutsukan.github.io リポジトリを作成
    - リポジトリのSettings > Pages で公開設定
- Astroでサイトを作成
